Setting up a new module project is a relatively easy process. We have already scaffolded out the code base and installed all the required dependencies. SimMod includes the latest version of SimCore, so there are only a few simple steps to get everything you need.

Step 1: Clone the master SimMod

Get the project from our GitHub repository located at https://github.com/Lumina-Datamatics-Inc/SimMod.
The URL for git is https://github.com/Lumina-Datamatics-Inc/SimMod.git

Make sure to choose a location on your hard drive that is easy to access from a terminal or command prompt, because you will be accessing this location on a regular basis.

After you have cloned the SimMod code to your computer, make sure to rename directory to reflect the name of the module you are going to work on. IE havc_4.

Step 4: Running the project in dev mode

SimMod projects run on node's command-line web sever. Compared to other web servers that you may be familiar with such as MAMP or WAMP, the node server uses Hot Reload. Hot Reload is the "hottest" new technology in the JavaScript world. This allows you to save your files and have the web browser update in real-time.

To start the server type the following command: npm run dev

Mac users:
Joes-MBP: hvac_4 joelomoglio$ npm run dev

Windows users:
C:\Users\joelomoglio\documents\www\hvac_4> npm run dev

If the build was successful, you will see the following output
