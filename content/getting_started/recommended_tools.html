ATOM Editor:

You can use any IDE or Text Editor of your choice, but we highly recommend using the Atom Editor. Atom is a text editor that's modern, approachable, yet hackable to the core, a tool you can customize to do anything but also use productively without ever touching a config file.



You can download and get instructions for installing Atom from their official website located at https://atom.io/

The following is a list of plugins we use to make development easier:

  2. atom-json-editor 0.4.1 | LukasHechenberger
  3. autoclose-html 0.23.0 | mattberkowitz
  4. autocomplete-json 5.5.0 | bali182
  5. abusy-signal 1.4.3 | steelbrain
  6. file-icons 2.1.10 | file-icons
  7. language-vue-component 0.5.0 | CYBAI
  8. linter 2.2.0 | steelbrain
  9. linter-ui-defult 1.6.5 | steelbrain
  10. vue2-automcomplete 0.1.0 | ealves-pt
  11. atom-file-icons 0.6.0 | donovanhiland

You can add as many or as little of these as you prefer. #7 is highly recommended for use with Vue files. Our code base uses a strict set of linting rules, so I would strongly suggest installing #8 & #9 which will give you real-time linting feedback. If your code has linting errors it will not compile and run.

SourceTree:

A free Git client for Windows and Mac. SourceTree simplifies how you interact with your Git repositories so you can focus on coding. Visualize and manage your repositories through SourceTree's simple Git GUI.



You can download a copy from their website at https://www.sourcetreeapp.com/
