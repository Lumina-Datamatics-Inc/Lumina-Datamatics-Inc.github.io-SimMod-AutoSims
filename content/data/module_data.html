Module data files will be generated by the SimGen data tool. At present time the SimGen tool has not been built yet, so the data files will manually need to be populated with data. This section will outline each file so that you understand what each is used for and what data you need to provide.

module-data.json

This file provides the majority of the data for the core module.


startup-screens-data.json

This file contains all the data for the Intro & Safety screens.


eoa-data.json

This file contains data that is required for the end of each activity.


review-questions-data.json

This file contains all the questions and answers for the review questions.


service-info-data.json

This file provides the data for populating the Service Information menu and popup window.


task-list-data.json

This file contains all the task data that is used to populate the Task List window.
