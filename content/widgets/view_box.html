Every view file can only have a single root element. The ViewBox is the view's root element. SimCore uses an activity view container to render each activity view, basically it grabs all the contents of the ViewBox and renders it to the view container.

The ViewBox widget requires an id and a background image. The id is the same as the name you provide for the view module, with the exception that its all lowercase. For the :background, you need to add the backgroundImg data property and the image file name.
