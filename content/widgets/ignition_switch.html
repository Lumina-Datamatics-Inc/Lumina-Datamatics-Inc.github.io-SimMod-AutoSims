The IgnitionSwitch widget is a base widget, we have tried to provide the most common functionality that we could. There will be cases where you will need different functionality, so we have made the code for this widget accessible so that you can copy it and makes changes as you need to.

How to use (example)

A1V2.vue

- Add tool to template in `conent-box`
  1. `isClickable` is necessary to correctly display errors when switch is locked
  2. `@incorrect` is an event fired when key is turned to wrong position
  3. Showing hints depends on currentAttempts passed to switch
  4. import widget in script part
  5. import IgnitionSwitch from '../../widgets/ignition-switch/IgnitionSwitch'`

a1v2-seq.js

- Enables switch and sets correct mode that needs to be chosen, callback is a function fired up after turning key to correct mode, additionalCallback is optional, when you want to fire a function during engine sound `this.ignitionNextMode = { val: 'lock', callback: this.t3a2, additionalCallback: this.t3ign }`
