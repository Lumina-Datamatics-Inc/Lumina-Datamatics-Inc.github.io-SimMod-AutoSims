The IgnitionSwitch widget is a base widget, we have tried to provide the most common functionality that we could. There will be cases where you will need different functionality, so we have made the code for this widget accessible so that you can copy it and makes changes as you need to.

I tried to make this widget as easy as possible to use by encapsulating the bulk of its functionality within the widget.

Lines 4 - 7: You set the position of the ignition-switch by setting the values of its four css position properties. You only need to set the ones that you need. Internally its set to an absolute position.

The ignition-switch has four positions Lock (Loc), Accessory (Acc), On and Start.

Lines 8 - 11: are events that are fired when you rotate the switch to each position. So you can call whatever sequence task action method you need to for what you are working on.

Lines 12 - 14: are the hints for the switch. You need to set the ones your not using to false and use the task ShowHint property for task that the hint is for.

The difficult part of any web based widget is having a event perform something different for various tasks. The best approach to modify this would be to pass in an array of task methods and hints and have them fire based on the current task.
