The Hotspot widget is used to create a clickable area over a background image or part of another image element. It requires an id and the :currentTask attribute must contain the currentTask data property, you must also provide the task attribute with the task that this goes to. The last requirement is to give the @action event the sequence task method to call when it's clicked.

The :currentTask attribute and the task attribute are used to by the widget to compare if the HotSpot should be clickable. This makes it so it can only be clicked when the currentTask is equal to the task that this belongs to.
The Hotspot doesn't allow for any content to be placed between its opening and closing tags. Anything placed between the tags will be ignored.

The HotSpot requires you to position it on the screen. You will need to create styles for each individual HotSpot. You will need to position this above a HintBox, the HotSpot must be at a higher z-index or it wont be able to be clicked.
