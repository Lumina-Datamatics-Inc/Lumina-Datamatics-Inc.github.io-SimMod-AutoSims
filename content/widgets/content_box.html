The ContentBox is the main area where you place all your activity html elements. It's also responsible for handling incorrect clicks. Students typically click on elements as part of completing a task. They will either click on the element that is correct, or they will click elsewhere, the ContentBox listens for click events and if its :isClickable attribute is set to true, it will emit the @action event which calls the incorrect() method.

By default isClickable needs to be set to false.

The incorrect method takes care the users attempts, score (points) and the incorrect response. It also updates the current task's hint value to true. The HintBox is the next topic.
