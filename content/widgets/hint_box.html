The HintBox is is used to show the student where the correct place to click is located. The HintBox requires an id for adding a css style for positioning it on the screen. The naming convention for the id is the task perpended to -hint.

It also has a v-hint directive that requires a boolean data property. You must follow this naming convention as well or it will not work properly. Prepend the task to ShowHint to create the property name.

When this data property is set to true, the HintBox will display, it has a yellow border and blinks, this is provided as part of this widget.

The HintBox doesn't allow for any content to be placed between its opening and closing tags. Anything placed between the tags will be ignored.

By default t1ShowHint needs to be set to false. You should set it to true when you need to position it on the screen.

The HintBox requires you to position it on the screen. You will need to create styles for each individual HintBox that you use. There will be situations where you will have to position this beneath another widget such as the HotSpot, or an clickable image the HintBox must be at a lower z-index or it will block the click event.
