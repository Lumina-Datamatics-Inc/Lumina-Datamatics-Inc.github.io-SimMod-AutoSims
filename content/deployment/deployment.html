When you have completed a module and you need to push it for testing, or for production. You will need to do build. You can't upload the raw files you have been working on, they need to be complied and transformed into ES5 Javascript files in order to run on a standard web server.

To build the project for deployment, type the following command at your terminal or command prompt:

$ npm run build

This will create a dist folder at the root evel of your project folder. Copy the contents of this folder and upload to whichever server that you need to.
