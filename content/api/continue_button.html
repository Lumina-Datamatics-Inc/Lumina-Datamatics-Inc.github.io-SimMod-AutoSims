The Continue Button is a common feature of SimMod. It's used to go from Activity View to Activity View and from Activity to Activity. The SimCore API provides you with 5 methods for using it.

  1. show()
  2. hide()
  3. next(view)
  4. end()
  5. callback(callback)

#1: The show() method will display the Continue Button. It has a fixed location, and there are no styles to set for it.

#2: The hide() method hides the Continue Button.

#3: The next(view) method is used when you need to display another Activity View. The view parameter is the ID of the view you want to load. For example 'A1V2'.

#4: The end() method is used to call the End of Activity screen. Calling this method also triggers SimCore to handle internal functions for ending the activity, such has processing the score, resetting global UI state and handling reporting data.

#5: The callback(callback) method is required. By default it hides the button, calls the end() method and you can also call any methods you need to in order to complete an activity, such as hiding an assessment.
