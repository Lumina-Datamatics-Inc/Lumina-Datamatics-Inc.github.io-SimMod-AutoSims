SimCore provides an API with functions for working with Global SimMod components. One of the major components you need to work with is the IconBar. The IconBar is comprised of a set of fixed required icons, along with a set of optional ones. The API methods deal with the optional icons.

IconBar Object

This object currently contains four properties:
  1. Zoom
  2. ServiceInfo
  3. ToolTray
  4. PartsDepartment

All these properties have four methods that all work the same.
  1. show()
  2. enable()
  3. disable()
  4. hide()

The first and last methods do not need an explanation. The second and third methods affect the state of the icons. When an icon is disabled it is not clickable and the icon is grayed out. When an icon is enabled it can be clicked and it changes to a white icon. Below shows the icons states.



To access these methods you call them like this:

this.$core.IconBar.PropertyName.method()
